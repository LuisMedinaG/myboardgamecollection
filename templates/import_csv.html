How to get the CSV from BoardGameGeek
  1. Open your collection on BGG: {{ if .BGGUsername }} boardgamegeek.com/collection/user/{{ .BGGUsername }} {{ else }} boardgamegeek.com/collection/user/<username> — set your BGG username on the Sync page to get a direct link {{ end }}
  2. In the top-right of the collection table, look for Download board games: followed by All and Owned. Click whichever you want to import.
  3. BGG will queue the export. After a few seconds you’ll receive a GeekMail with a link to the generated .csv file. Download it.
  4. Upload that file below, click Preview to see what will be imported, then confirm the import.

The only required column is objectid (the BGG game ID). objectname is recommended for the preview.
